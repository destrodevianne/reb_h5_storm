Премиум Системная учетная запись
Информация и выгоды от премиум системе счета.
 

Как только счет производится премии, каждый персонаж на этот счет пойдет на пользу бонусы премиум аккаунт может предложить. Имея премиум аккаунт, не делает вас больше, чем специальные остальных игроков. Вы по-прежнему рассматриваться как обычный плеер по команде GM / Админ.
Серебряные Бонусы данные
- Exp: +30%
- SP: +30%
- Можно использовать команду .whereis
Золото данные Бонусы
- Exp: +50%
- SP: +50%
- Adena: +30%
- Можно использовать команду .whereis
- Доступ GMhop за городом.
Платина данные Бонусы
- Exp: +100%
- SP: +100%
- Adena: +50%
- Можно использовать команду .whereis
- Доступ GMhop за городом.
- Бесконечность Shoots / Стрелки.
- Доступ к буферной за городом..

Легенда: .whereis (целевое имя) - показывает расположение + квест указатель другу, член партии или член клана. Использование GMShop за пределами города, только если игрок не в пвп, состояние боя, мертвы, олимпиады, событий и других нормальных безопасных проверок. Использование буфера за пределами города, только если игрок любители ниже 2 минуты и его не в ограничениях Осада / эпических / зоны экземпляр + GMShop. Бесконечность стрелки / выстрелы, необходимо иметь по крайней мере 1 в вашем инвентаре. Статистика Показать игрока - способность менять кнопкой мыши на игроков, чтобы посмотреть их общие статистику (уровень, класс, HP, MP, CP, ПВП убийств, ПК убийств, баффов).
